Windows Event Log Js

Native node.js module to log messages to the Windows EventLog

Installation

$ npm install windows-eventlog

Usage

Initialize somewhere the logger like:

  var EventLog = require('windows-eventlog').EventLog;
  var myeventlog = new EventLog("mySource");
  myeventlog.log("a message");

And you will see this:

new EventLog(source[, logName])

This create an instance of the EventLog with the given source. You can optionally pass a logName, defaults to "Application".

If the source doesn't exist in the event log database it will be created with the givne log name.

eventLog.log(message[, logEntryType])

This method will create an entry in the event log with the given message.

Optionally you can specify a logEntryType. The possible values for logEntryType are "Information", "Warning", "Error" and others two that you will never use :)

How it works

This module was built on c++/cli (.Net) and uses System.Diagnostics.EventLog.

In order to log events you need to run the application with an elevated account: ie administrator or system account. Windows services typically run under the system account, if you are looking on how to run node.js applications as a Windows Service have a look to WinSer.
